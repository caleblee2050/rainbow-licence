# M2 시연 시나리오 — 수동 체크리스트

T20 배포 후 https://rainbow-licence.vercel.app 에서 다음 4개 시나리오 검증.

## 시나리오 A — 시연 모드 신규 학생 (vi)

- [ ] localStorage 비운 상태로 진입
- [ ] AuthPage에서 학교 코드 `NEXT_SCHOOL` + 베트남어 선택 + 시작
- [ ] 한식조리 자격증 진입 → "내 자료" 탭
- [ ] 짧은 한국어 식품위생 텍스트(예: "식품위생 HACCP 7원칙 ...") 붙여넣고 등록
- [ ] SourceDetailPage 자동 진입, 단계별 status 진행 확인 (~30~60초)
- [ ] done 후 미리보기에서 개념·문제·베트남어 번역 확인
- [ ] 학습 화면으로 돌아가서 "내 자료 N문제" chip 클릭 → 사용자 문제 풀이
- [ ] STEP 1에서 한국어+베트남어 병행, STEP 2에서 키워드 점선 밑줄, STEP 3에서 ❓ 슬라이딩 패널 작동

## 시나리오 B — 이메일 로그인 (다기기 시뮬레이션)

- [ ] 시나리오 A 마친 상태에서 이메일 모드로 이메일 입력
- [ ] 매직 링크 클릭 → /api/auth/verify → 메인 화면 redirect
- [ ] 다른 브라우저에서 동일 이메일로 매직 링크 받기 → 동일 자료 보이는지 확인

## 시나리오 C — PDF 업로드

- [ ] 작은 한국어 PDF 준비
- [ ] PDF 모드로 등록
- [ ] 텍스트 추출 → 파이프라인 진행 확인
- [ ] 스캔 이미지 PDF 시도 → 명확한 에러 메시지 (PdfExtractError)

## 시나리오 D — 무관 자료 도메인 분류

- [ ] 한식조리 자격증에 무관 텍스트(예: 운전 매뉴얼) 등록
- [ ] SourceDetailPage에서 `domain_score < 0.4` 경고 표시 확인
- [ ] 처리는 진행되지만 결과 문제 품질 낮음
